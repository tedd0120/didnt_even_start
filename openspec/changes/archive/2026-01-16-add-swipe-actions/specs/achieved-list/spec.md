## 修改需求
### 需求：达成清单页面
系统必须在“临时上进”页面顶部展示主标题，副标题紧随其后且不重复显示主标题。

#### 场景：进入达成清单
- **当** 用户进入“临时上进”页面
- **那么** 系统在页面顶部显示主标题“临时上进”
- **并且** 主标题下方仅显示动态副标题

## 新增需求
### 需求：达成事项管理
系统必须允许用户通过列表右滑手势对达成事项进行置顶与删除（弹窗确认）。

#### 场景：右滑操作
- **当** 用户在达成清单中对某事项右滑
- **那么** 系统展示置顶与删除操作

#### 场景：删除确认
- **当** 用户选择删除操作
- **那么** 系统弹出确认提示并在确认后删除该事项
