# badge-wall 规范增量

## 新增需求
### 需求：双界树物种概览
系统必须在勋章墙页面展示“双界树已解锁物种”的概览卡片。

#### 场景：查看物种概览
- **当** 用户进入勋章墙页面
- **那么** 系统展示海底与天空已解锁物种数量

### 需求：分享内容选择
系统必须允许用户在勋章墙页面选择分享“勋章拼图”或“已解锁物种”。

#### 场景：切换分享类型
- **当** 用户在分享区域选择“勋章”或“物种”
- **那么** 系统切换对应的分享预览内容

#### 场景：分享已解锁物种
- **当** 用户选择“物种”并点击保存
- **那么** 系统将已解锁物种拼图保存到相册

### 需求：海报头像昵称开关一致性
系统必须让“海报显示头像与昵称”的开关同时作用于勋章海报与物种海报。

#### 场景：物种海报显示头像昵称
- **当** 用户开启“海报显示头像与昵称”并切换至物种海报
- **那么** 系统在物种海报中显示头像与昵称
