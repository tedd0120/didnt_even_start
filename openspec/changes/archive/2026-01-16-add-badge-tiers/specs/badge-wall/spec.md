## 修改需求
### 需求：勋章墙展示
系统必须在勋章墙展示普通勋章与隐藏勋章的状态与等级，并对隐藏勋章使用占位显示。

#### 场景：进入勋章墙
- **当** 用户进入勋章墙页面
- **那么** 系统展示所有普通勋章的名称、等级、图标与达成条件
- **并且** 未解锁的普通勋章以弱化样式展示
- **并且** 隐藏勋章显示为“？？？”占位，并隐藏其名称与达成条件

### 需求：勋章解锁规则
系统必须按放弃事项数量解锁普通勋章与隐藏勋章，并记录其等级。

#### 场景：解锁普通勋章（条件公开）
- **当** 用户累计放弃事项达到以下条件
- **那么** 系统解锁对应普通勋章并标记其等级
  - 铜：入门躺平奖（1 条）
  - 铜：拖延起步者（3 条）
  - 铜：计划搁置者（5 条）
  - 银：咸鱼进阶者（10 条）
  - 银：沙发守护者（20 条）
  - 银：懒人代表作（30 条）
  - 金：躺平大师（50 条）
  - 金：省力宗师（80 条）
  - 金：梦想搁置王（120 条）

#### 场景：解锁隐藏勋章（条件隐藏）
- **当** 用户累计放弃事项达到以下条件
- **那么** 系统解锁对应隐藏勋章并标记其等级
- **并且** 在勋章墙内不显示其名称与达成条件
  - 铂金：隐秘一号（200 条）
  - 铂金：隐秘二号（300 条）
  - 钻石：隐秘三号（500 条）
  - 钻石：隐秘四号（800 条）
  - 暗物质：隐秘五号（1200 条）
  - 暗物质：隐秘六号（2000 条）
