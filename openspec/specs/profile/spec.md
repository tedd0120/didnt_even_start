## 需求
### 需求：个人资料编辑
系统必须提供“我”页面用于编辑头像与昵称，并将资料持久化到本地存储。

#### 场景：进入个人资料页
- **当** 用户进入“我”页面
- **那么** 系统展示当前头像与昵称，并提供编辑入口

#### 场景：从相册选择头像
- **当** 用户选择从相册更新头像
- **那么** 系统请求相册权限并在授权后更新头像

#### 场景：使用拍照更新头像
- **当** 用户选择拍照更新头像
- **那么** 系统请求相机权限并在授权后更新头像

#### 场景：更新昵称
- **当** 用户输入并保存新昵称
- **那么** 系统保存昵称并在下次打开时仍可显示

### 需求：头像展示样式
系统必须以圆形头像展示用户图片，并添加柔化边框以减弱裁切生硬感。

#### 场景：显示头像
- **当** 用户设置了头像
- **那么** 系统以圆形裁切显示头像并带有柔化边框
