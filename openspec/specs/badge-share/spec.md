# badge-share 规范

## 目的
明确勋章拼图分享入口的行为、拼图内容的组成要素与保存到相册的流程，并定义保存成功或失败时的用户反馈要求。
## 需求
### 需求：勋章拼图分享
系统必须生成排版紧凑的勋章拼图图片，以减少海报空白区域。

#### 场景：紧凑排版
- **当** 系统生成勋章拼图图片
- **那么** 海报的标题区、勋章区与页脚区采用紧凑间距

### 需求：保存结果反馈
系统必须在保存成功或失败时提示用户结果。

#### 场景：保存成功
- **当** 勋章拼图成功保存到相册
- **那么** 系统提示保存成功

#### 场景：保存失败
- **当** 勋章拼图保存失败或被拒绝权限
- **那么** 系统提示失败原因并告知可在设置中授权
